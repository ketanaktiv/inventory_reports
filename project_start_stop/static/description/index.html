Project Task Start Stop

Basic Features
  • Using this app user can easily manage timesheet on task.
  • Automatically add an entry in timesheet at task start.
  • Calculate total time spent between the start and stop of any task.
  • Can't assign task to another user in working state.
  • Can't start two tasks at the same time.
  • Can't start/stop task assigned to another user.
